# Projeto Speed

Este arquivo descreve a organização dos diretórios e a estrutura do projeto Speed.

### Descrição das Pastas

* **`MikroC/`**
    * Contém todo o código-fonte (`.c`, `.h`) a ser compilado e gravado no microcontrolador, referente a IDE MikroC.

* **`Proteus/`**
    * Contém o arquivo de projeto do Proteus (`.pdsprj`) e outros arquivos relacionados à simulação do circuito.

* **`Docs/`**
    * Pasta destinada a toda a documentação do projeto, como diagramas, esquemáticos, *datasheets* e anotações.
